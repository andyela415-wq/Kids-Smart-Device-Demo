# 儿童智能硬件屏幕 Demo

320×240 横屏嵌入式学习设备 UI 模拟器，面向 6–12 岁儿童学习机/电子词典风格界面。

## 技术栈

- React 19
- React Router
- Vite
- 纯 CSS（320×240 固定逻辑分辨率）

## 快速开始

你的 Mac 上如果没有安装 Node.js / npm，请用项目自带的脚本（推荐）：

```bash
bash setup.sh   # 首次运行：下载 Node 并安装依赖
bash dev.sh       # 启动开发服务器
```

浏览器打开 **http://127.0.0.1:5173/**

若系统已安装 Node.js，也可以直接使用：

```bash
npm install
npm run dev
```

> 注意：请分三行执行命令，不要连在一起粘贴（例如 `Demo npm install` 会报错）。

## 项目结构

```
src/
├── components/       # 通用 UI 组件
├── layouts/          # 设备布局
├── pages/            # 路由页面（占位）
├── router/           # 路由配置
└── styles/           # 全局样式与设计变量
```

## 屏幕规格

| 属性     | 值        |
| -------- | --------- |
| 逻辑分辨率 | 320 × 240 |
| 方向     | 横屏       |
| 缩放     | 自适应放大  |

所有 UI 按 320×240 像素设计，外层 `DeviceFrame` 会根据浏览器窗口等比缩放，便于在电脑上预览硬件效果。
